# 🌍 Global Protest Map

An interactive web application for tracking and visualizing protests worldwide using Leaflet.js and MongoDB.

## Features
- Interactive global map
- Protest location markers
- Country and category filtering
- Real-time style visualization
- Responsive UI

## Tech Stack
- Next.js
- React
- TypeScript
- MongoDB
- Leaflet.js
- Tailwind CSS

## Installation

```bash
npm install
npm run dev
